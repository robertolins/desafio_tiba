# Desafio Tiba
=====================

Instalação
------------
    * Instalar as dependências do FrontEnd (/client) e do BackEnd (/server)
    * Copiar e renomear o arquivo .env.development para .env
    * Adicionar as configurações do banco de dados no arquivo .env

Execução
------------
    # No diretório /server
      npm start
    
    
    # No dietório /client
      npm start
